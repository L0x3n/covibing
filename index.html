A simulated preview running in your browser. The real app runs actual agents on your machine.

🧩 Up to 16 agent terminals

Split and resize in any direction. Claude Code, Codex and any shell command, each in its own pane with git branch, unread badges and a prompt queue.

👀 Watch your friends live

Share a pane to your session and friends see the agent's terminal in real time — view-only, always. Nobody can ever type into your machine.

🔑 Your own AI accounts

Everyone signs in to their own Claude/Codex on their own machine. Covibing never sees, stores or proxies your AI credentials or code.

⚡ Peer-to-peer streaming

Terminal streams go directly between machines over WebRTC. Our server only handles accounts, sessions and invites.

🎯 Drag-and-drop skills

Keep a library of agent skills and drop them onto any pane to install into that project's .claude/skills/.

🪶 A 1.5 MB install

Built on Rust + Tauri, not Electron. The whole installer is a few megabytes and needs no admin rights — it uses the WebView already on your PC.

Privacy by architecture

Terminal output, scrollback, project paths and layouts never touch our servers — they live on your machine and stream peer-to-peer only to session members you invited. What we store: your email, display name, session names and invite codes. That's it.

Windows says "Windows protected your PC"?

The beta installer isn't code-signed yet, so SmartScreen shows a warning the first time. Click More info → Run anyway. The download always comes from our GitHub Releases page, so you can verify the source yourself.
